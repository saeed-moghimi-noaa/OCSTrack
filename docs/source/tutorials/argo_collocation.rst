Argo Float 3D Collocation Tutorial
===================================

This tutorial demonstrates collocating Argo float temperature and salinity profiles 
with 3D ocean model outputs.

Overview
--------

In this tutorial, you will:

1. Download Argo float data
2. Load 3D model outputs with vertical coordinates
3. Perform vertical interpolation and collocation
4. Analyze profile comparisons

Prerequisites
-------------

* OCSTrack installed
* Internet connection (for data download)
* 3D model outputs (temperature, salinity, vertical coordinates)
