Satellite Altimetry Collocation Tutorial
=========================================

This tutorial demonstrates a complete workflow for collocating satellite altimetry 
data with ocean model outputs.

Overview
--------

In this tutorial, you will:

1. Download satellite altimetry data from NOAA CoastalWatch
2. Load SCHISM model outputs
3. Perform spatial and temporal collocation
4. Analyze and visualize results

Prerequisites
-------------

* OCSTrack installed
* Internet connection (for data download)
* SCHISM model outputs available locally
